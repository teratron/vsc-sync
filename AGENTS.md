# Agent Rules

## 1. Language Policy

Consistency in communication and code is paramount.

### 1.1 Code & Technical Elements (English)

- **Identifiers**: All variables, functions, classes, methods, and types.
- **Environment**: Error messages, logs, and API definitions.

### 1.2 Documentation & Communication (Russian)

- **Comments**: All inline code comments and docstrings.
- **Documentation**: Technical guides, READMEs, and implementation notes.
- **Process**: Commit messages, pull request descriptions, and issue titles.
- **Interaction**: Chat discussions, explanations, and strategic planning.

## 2. Completion Protocol (Mandatory Checklist)

Before finishing any task, the agent MUST verify the following:

- [ ] **Technical Language**: All code identifiers and system logs are in English.
- [ ] **Documentation & Communication**: All comments, docs, and chat responses are in Russian.
